# Investment Recommendation Advisor 

A rule-based expert system for personalized investment portfolio recommendations using **60+ AI rules**.

## Tech Stack
- **Backend:** Python · FastAPI
- **Frontend:** Streamlit
- **Database:** MySQL
- **Visualization:** Plotly

## Features
- 65+ investment rules across 15 categories
- Priority-based inference engine with certainty factors
- Full explainability — every rule traced
- Risk assessment with score (0-100)
- Portfolio allocation across 6 asset classes
- Scenario testing (what-if analysis)
- User authentication with JWT
- Recommendation history
